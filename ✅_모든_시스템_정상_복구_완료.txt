Q: 웹이 느리거나 안 열립니다
A: 1. 서버 재시작:
      taskkill /F /IM node.exe
      START_ALL.bat
   2. 브라우저 캐시 삭제
   3. 강력 새로고침 (Ctrl + Shift + R)

Q: APK에서 "Connection refused"
A: 1. 백엔드 서버 실행 확인
   2. 웹 서버 실행 확인
   3. 서버 URL 확인
   4. 방화벽 확인

Q: 스타일이 이상합니다
A: 브라우저 캐시 삭제 후 강력 새로고침

Q: 데이터가 안 보입니다
A: 1. 백엔드 서버 실행 확인
   2. API 연결 확인 (F12 콘솔)
   3. 네트워크 탭에서 오류 확인
