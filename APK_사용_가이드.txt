========================================
  📱 펌피 APK 외부 접속 가이드
========================================

✅ 최신 APK 파일: Pumpy_External_Access.apk

========================================
  🚀 설치 및 실행 방법
========================================

1. APK 설치
   - Pumpy_External_Access.apk를 휴대폰으로 전송
   - 설치 진행 (보안 설정 허용 필요)

2. 서버 터널 시작
   - 컴퓨터에서 START_TUNNEL.bat 실행
   - 2개의 터널 창이 열림 (닫지 마세요!)
   
3. APK 실행 및 설정
   - 앱을 실행하면 "서버 설정" 화면이 나타남
   - 서버 URL 입력:
     
     https://pumpy-app-2025.loca.lt
   
   - "저장 및 연결" 버튼 클릭

4. 완료!
   - 앱이 로드되고 모든 기능 사용 가능
   - 어느 네트워크에서든 접속 가능!

========================================
  📡 서버 URL 종류
========================================

1. 같은 Wi-Fi (로컬):
   http://172.30.1.44:3000
   
2. 외부 네트워크 (LocalTunnel):
   https://pumpy-app-2025.loca.lt

========================================
  ⚠️ 주의사항
========================================

1. LocalTunnel 사용 시:
   - START_TUNNEL.bat가 실행 중이어야 함
   - Frontend Tunnel, Backend Tunnel 모두 필요
   - 터널 창을 닫으면 연결 끊김

2. 처음 접속 시:
   - LocalTunnel 보안 화면이 나타날 수 있음
   - "Click to Continue" 버튼 클릭
   - 또는 외부 IP 주소 입력 (한 번만)

3. 연결이 끊긴 경우:
   - START_TUNNEL.bat 재실행
   - 앱에서 설정 페이지로 가서 URL 재입력

========================================
  🔧 문제 해결
========================================

Q: 로딩 중에서 멈춰있어요
A: 1. 터널이 실행 중인지 확인
   2. 서버 URL이 정확한지 확인
   3. 앱을 완전히 종료 후 재실행

Q: "연결할 수 없습니다" 오류
A: 1. START_TUNNEL.bat 실행 확인
   2. Frontend 서버 실행 확인 (포트 3000)
   3. Backend 서버 실행 확인 (포트 8000)

Q: 데이터가 보이지 않아요
A: 1. Backend API가 정상 작동하는지 확인
   2. 브라우저에서 먼저 테스트
      https://pumpy-api-2025.loca.lt/api/members/
