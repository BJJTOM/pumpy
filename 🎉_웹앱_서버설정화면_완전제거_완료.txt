만약 여전히 서버 설정 화면이 나타난다면:

1️⃣ **브라우저 캐시 완전 삭제**
   - Chrome: Ctrl + Shift + Delete
   - 전체 기간 선택
   - "캐시된 이미지 및 파일" 체크
   - "데이터 삭제" 클릭

2️⃣ **localStorage 수동 삭제**
   - F12 (개발자 도구 열기)
   - Application 탭
   - Local Storage → http://localhost:3000
   - 모든 항목 삭제 (특히 'serverUrl')

3️⃣ **하드 새로고침**
   - Ctrl + Shift + R (Chrome)
   - Ctrl + F5 (Firefox)

4️⃣ **새 탭에서 다시 열기**
   - http://localhost:3000/auth/login
   - 로그인
   - http://localhost:3000/app 접속
