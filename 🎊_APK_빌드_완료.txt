📖 Android Studio 개발 방법:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Android Studio 설치
   https://developer.android.com/studio

2. AVD (에뮬레이터) 생성
   Tools → Device Manager → Create Device

3. 개발 서버 실행
   cd pumpy-app
   npm start

4. 에뮬레이터에서 앱 실행
   터미널에서 'a' 키 입력

5. 코드 수정 → 자동 반영! ⚡
   - Fast Refresh 활성화
   - 저장하면 즉시 반영
   - 빌드 필요 없음!
