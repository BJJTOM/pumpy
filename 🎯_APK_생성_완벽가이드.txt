📱 APK 생성 방법 (3단계)
------------------------------------------------------------------

방법 A: PWA Builder (5분, 가장 간단!)
방법 B: Vercel + PWA Builder (10분, 추천!)

==================================================================

🚀 방법 A: PWA Builder 직접 사용
------------------------------------------------------------------

1. 앱 아이콘 준비
   - 512x512 PNG 이미지 준비
   - https://realfavicongenerator.net/ 에서 자동 생성
   - 생성된 아이콘들을 gym_web/public/icons/ 에 저장

2. 로컬 서버 실행
   cd gym_web
   npm run build
   npm start

3. Ngrok으로 임시 공개 URL 생성
   ngrok http 3000
   
   → 생성된 URL 복사 (예: https://abc123.ngrok.io)

4. PWA Builder 사용
   - https://www.pwabuilder.com/ 접속
   - URL 입력: https://abc123.ngrok.io/app
   - [Start] 클릭
   - [Package For Stores] 클릭
   - [Android] 선택
   - 옵션 설정:
     * Package ID: com.pumpy.gym
     * App name: 펌피
     * Launcher name: 펌피
     * Version: 1.0.0
   - [Generate] 클릭
   - APK 다운로드!

==================================================================

⭐ 방법 B: Vercel 배포 + PWA Builder (추천!)
------------------------------------------------------------------

1. GitHub에 코드 푸시
   git add .
   git commit -m "Add PWA support"
   git push

2. Vercel 배포 (무료!)
   - https://vercel.com/ 가입
   - [New Project] 클릭
   - GitHub 레포지토리 선택
   - Framework: Next.js 선택
   - Root Directory: gym_web
   - [Deploy] 클릭
   - 배포 완료! (예: https://your-app.vercel.app)

3. PWA Builder 사용
   - https://www.pwabuilder.com/ 접속
   - URL 입력: https://your-app.vercel.app/app
   - [Start] 클릭
   - [Package For Stores] 클릭
   - [Android] 선택
   - APK 다운로드!

4. 테스트
   - APK를 안드로이드 기기로 전송
   - 설치 및 실행
   - 완벽하게 작동! 🎉

==================================================================

🔧 방법 C: Capacitor 사용 (고급)
------------------------------------------------------------------

1. Capacitor 설치
   cd gym_web
   npm install @capacitor/core @capacitor/cli
   npm install @capacitor/android

2. Capacitor 초기화
   npx cap init pumpy com.pumpy.gym --web-dir=out

3. Android 프로젝트 생성
   npx cap add android

4. Next.js 정적 빌드
   - next.config.js 수정:
     
     module.exports = {
       output: 'export',
       images: {
         unoptimized: true
       }
     }

   npm run build

5. Capacitor 동기화
   npx cap sync

6. Android Studio에서 빌드
   npx cap open android
   
   Android Studio에서:
   - Build > Build Bundle(s) / APK(s) > Build APK
   - app/build/outputs/apk/debug/app-debug.apk 생성됨

🎨 앱 아이콘 생성 방법
------------------------------------------------------------------

온라인 도구 사용 (추천):

1. 이미지 준비
   - 1024x1024 PNG (정사각형)
   - 투명 배경 또는 단색 배경
   - 앱을 대표하는 심플한 디자인

2. 자동 생성 사이트
   https://realfavicongenerator.net/
   
   - [Select your Favicon image] 클릭
   - 이미지 업로드
   - Android Chrome 옵션:
     * Theme color: #667eea
     * Name: 펌피
   - [Generate] 클릭
   - [Download] 받기
   - 압축 해제 후 gym_web/public/icons/ 에 복사

3. 필요한 아이콘 크기
   - android-chrome-192x192.png → icon-192x192.png로 이름 변경
   - android-chrome-512x512.png → icon-512x512.png로 이름 변경
   - 나머지 크기들도 동일하게 생성

빠른 생성:
   https://icon.kitchen/
   - 아이콘 디자인 선택
   - 색상 설정
   - 다운로드

📦 APK 설치 방법
------------------------------------------------------------------

방법 1: USB 케이블
   1. 안드로이드 기기 설정 > 개발자 옵션 활성화
   2. USB 디버깅 활성화
   3. PC와 USB 연결
   4. 명령어 실행:
      adb install pumpy.apk

방법 2: 파일 전송
   1. APK를 기기로 전송 (카카오톡, 이메일 등)
   2. 다운로드 폴더에서 APK 클릭
   3. "알 수 없는 출처" 허용
   4. 설치 진행

방법 3: Google Drive
   1. APK를 Google Drive에 업로드
   2. 기기에서 Drive 앱 열기
   3. APK 다운로드 및 설치

방법 4: QR 코드
   1. APK를 웹 서버에 업로드
   2. QR 코드 생성
   3. 기기로 QR 코드 스캔
   4. 다운로드 및 설치

🏪 Play Store 배포 (선택사항)
------------------------------------------------------------------

1. 계정 준비
   - Google Play Console: https://play.google.com/console
   - 일회성 등록비: $25 (평생 유효)

2. 앱 등록
   - [Create app] 클릭
   - App name: 펌피
   - Default language: 한국어
   - App or game: App
   - Free or paid: Free

3. 앱 콘텐츠 작성
   - App icon (512x512)
   - Feature graphic (1024x500)
   - Screenshots (최소 2개)
     * Phone: 16:9 또는 9:16
     * Tablet (선택): 16:10 또는 10:16
   - Short description (80자)
   - Full description (4000자)

4. Store settings
   - Category: Health & Fitness
   - Contact details
   - Privacy policy URL (필수)

5. APK 업로드
   - Production > Create new release
   - Upload APK or AAB
   - Release notes 작성
   - Review and rollout

6. 심사 대기
   - 보통 1-3일 소요
   - 승인되면 자동 배포

❓ 자주 묻는 질문
------------------------------------------------------------------

Q: APK 크기가 얼마나 되나요?
A: PWA Builder로 생성 시 약 5-10MB

Q: iOS 앱도 만들 수 있나요?
A: 네! PWA Builder에서 iOS 옵션 선택하면 됩니다.

Q: 업데이트는 어떻게 하나요?
A: 웹 앱을 업데이트하면 자동으로 반영됩니다.
   새 APK는 큰 변경사항이 있을 때만 필요합니다.

Q: 오프라인에서도 작동하나요?
A: 네! Service Worker가 캐싱 처리합니다.

Q: 푸시 알림을 보낼 수 있나요?
A: 네! Service Worker에 푸시 알림 기능이 포함되어 있습니다.

Q: 네이티브 기능 (카메라, GPS 등)을 사용할 수 있나요?
A: PWA는 제한적, Capacitor 사용 시 모든 네이티브 기능 가능
