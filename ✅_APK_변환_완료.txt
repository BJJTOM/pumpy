🚀 APK 생성 3단계
------------------------------------------------------------------

1단계: 아이콘 생성 (5분)
   ① http://localhost:3000/icon-generator.html 접속
   ② 각 크기별 버튼 클릭하여 다운로드
   ③ 다운로드한 파일들을 gym_web/public/icons/ 폴더에 저장

2단계: 앱 배포 (5분)
   방법 A: Vercel (추천!)
     ① https://vercel.com/ 가입
     ② GitHub 레포지토리 연결
     ③ gym_web 폴더 선택
     ④ Deploy 클릭
     ④ 배포 URL 복사 (예: https://your-app.vercel.app)
   
   방법 B: Netlify
     ① https://www.netlify.com/ 가입
     ② Drag & Drop 배포
     ③ URL 복사

3단계: APK 생성 (5분)
   ① https://www.pwabuilder.com/ 접속
   ② URL 입력: https://your-app.vercel.app/app
   ③ [Start] 클릭
   ④ [Package For Stores] 클릭
   ⑤ [Android] 선택
   ⑥ 옵션 입력:
      - Package ID: com.pumpy.gym
      - App name: 펌피
      - Version: 1.0.0
   ⑦ [Generate] 클릭
   ⑧ APK 다운로드!

총 소요 시간: 약 15분

🔧 문제 해결
------------------------------------------------------------------

Q: "manifest.json이 로드되지 않아요"
A: gym_web/public/ 폴더에 있는지 확인
   브라우저에서 /manifest.json 직접 접속해보기

Q: "Service Worker 등록 실패"
A: HTTPS 필요 (localhost는 예외)
   Vercel 배포하면 자동 해결

Q: "앱 아이콘이 안 보여요"
A: icons/ 폴더와 파일명 확인
   manifest.json의 경로 확인

Q: "PWA Builder에서 에러가 나요"
A: manifest.json 문법 확인
   https://web.dev/manifest-checker/ 에서 검증
