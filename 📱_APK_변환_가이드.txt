📱 회원 앱 APK 변환 가이드
==================================================================

🎯 3가지 방법
------------------------------------------------------------------

방법 1: PWA Builder (가장 간단!) ⭐ 추천
방법 2: Capacitor (네이티브 기능 필요시)
방법 3: React Native (완전한 네이티브 앱)

==================================================================

🚀 방법 1: PWA Builder (추천)
------------------------------------------------------------------

특징:
  ✅ 가장 빠르고 간단
  ✅ 기존 웹앱 그대로 사용
  ✅ 무료
  ✅ 5분 안에 APK 생성
  ✅ Play Store 배포 가능

단계:

1. PWA 설정 파일 추가
   - manifest.json (이미 생성 예정)
   - service-worker.js (이미 생성 예정)

2. PWA Builder 사용
   - https://www.pwabuilder.com/ 접속
   - 앱 URL 입력: http://your-domain.com/app
   - "Start" 클릭
   - "Package For Stores" 클릭
   - "Android" 선택
   - APK 다운로드

3. 설치 및 테스트
   - APK 파일을 안드로이드 기기로 전송
   - 설치 허용 (알 수 없는 출처)
   - 앱 설치 및 실행

==================================================================

⚡ 방법 2: Capacitor (중급)
------------------------------------------------------------------

특징:
  ✅ 네이티브 기능 접근 가능
  ✅ 카메라, GPS, 푸시 알림 등
  ✅ iOS도 가능
  ❌ 설정이 조금 복잡

단계:

1. Capacitor 설치
   npm install @capacitor/core @capacitor/cli
   npx cap init

2. Android 플랫폼 추가
   npm install @capacitor/android
   npx cap add android

3. 빌드 및 동기화
   npm run build
   npx cap sync
   npx cap open android

4. Android Studio에서 빌드
   - Build > Build Bundle(s) / APK(s)
   - Build APK
   - app/build/outputs/apk/ 에서 APK 찾기

📱 APK 테스트 방법
------------------------------------------------------------------

1. USB 디버깅
   - 안드로이드 기기 설정 > 개발자 옵션
   - USB 디버깅 활성화
   - PC와 연결
   - adb install app.apk

2. 파일 전송
   - APK를 기기로 전송
   - 파일 관리자에서 APK 클릭
   - 설치 허용

3. 웹사이트 배포
   - APK를 웹사이트에 업로드
   - 기기에서 다운로드
   - 설치

==================================================================

🏪 Play Store 배포
------------------------------------------------------------------

1. Google Play Console 가입
   - https://play.google.com/console
   - 25달러 일회성 등록비

2. 앱 등록
   - 새 앱 만들기
   - 앱 정보 입력
   - 스크린샷 업로드 (최소 2개)

3. APK 업로드
   - 프로덕션 트랙
   - APK 또는 AAB 업로드
   - 심사 제출

4. 심사 대기
   - 보통 1-3일 소요
   - 승인되면 자동 배포

⚠️ 주의사항
------------------------------------------------------------------

1. HTTPS 필수
   - PWA는 HTTPS에서만 작동
   - 로컬 테스트: localhost는 OK
   - 배포 시: SSL 인증서 필요

2. 도메인 필요
   - PWA Builder 사용 시 공개 URL 필요
   - 임시: Vercel, Netlify 무료 배포
   - 정식: 도메인 구매 + 호스팅

3. 앱 크기
   - PWA는 웹앱 캐싱
   - 첫 실행 시 다운로드
   - 오프라인 지원 가능

4. 업데이트
   - 웹 업데이트하면 자동 반영
   - 별도 APK 업데이트 불필요
   - Service Worker 캐시 관리
