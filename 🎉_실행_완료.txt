Expo 서버가 시작되었습니다!

📲 사용 방법:

1️⃣ Play Store에서 "Expo Go" 앱 설치
   https://play.google.com/store/apps/details?id=host.exp.exponent

2️⃣ 스마트폰과 컴퓨터를 같은 Wi-Fi에 연결

3️⃣ Expo 터미널에서 QR 코드 스캔
   (또는 터미널에서 링크 확인)

4️⃣ 앱 실행 후 서버 설정:
   - CMD에서 ipconfig 실행
   - IPv4 주소 확인 (예: 172.30.1.44)
   - 앱에 입력: http://172.30.1.44:3000

5️⃣ 저장 및 연결 테스트

6️⃣ 완료! 🎉

같은 Wi-Fi가 아닌 곳에서 사용하려면:

1. 새 터미널 열기

2. ngrok 실행:
   ngrok http 3000

3. ngrok URL 복사 (예: https://abc123.ngrok-free.app)

4. 앱의 서버 설정에 입력

5. 이제 어디서든 접속 가능!

Q: 웹이 열리지 않습니다
A: 몇 초 기다린 후 다시 시도하세요
   Next.js가 완전히 시작될 때까지 20~30초 소요

Q: 앱에서 "Connection refused"
A: 1. 백엔드가 실행 중인지 확인
   2. 서버 URL이 올바른지 확인
   3. 같은 Wi-Fi에 연결되어 있는지 확인

Q: Expo QR 코드가 보이지 않습니다
A: Expo 터미널 창을 확인하세요
   또는 터미널에 표시된 URL을 Expo Go 앱에 직접 입력

Q: IP 주소를 모르겠습니다
A: 새 CMD 창에서:
   ipconfig
   → IPv4 주소 확인
