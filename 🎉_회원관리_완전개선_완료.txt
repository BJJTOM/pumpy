✨ 주요 개선 사항
------------------------------------------------------------------

1. 리스트 형식으로 완전 재설계
   ❌ 이전: 카드 그리드 형식
   ✅ 현재: 테이블 리스트 형식
   
2. 데스크톱 & 모바일 반응형
   - PC: 테이블 형식 (7열)
   - 모바일: 카드 형식 (자동 전환)

3. 회원 상세 페이지 재설계
   - 프로필 아바타 추가
   - KPI 카드 4개
   - 탭 메뉴 (기본정보/출석/회원권/메모)
   - 빠른 액션 버튼

4. 추가된 기능
   - 📊 정렬 기능 (최신순/이름순/출석순)
   - 🔍 통합 검색 (이름/전화번호/이메일)
   - 📊 실시간 통계 (활성/대기/만료임박)
   - ✅ 출석 체크 버튼
   - 🎫 회원권 구매 모달
   - 📝 메모 작성 기능
   - 🔄 상태 변경 드롭다운

📊 회원 관리 페이지 (리스트 형식)
------------------------------------------------------------------

레이아웃:
┌───┬─────────┬────────────┬──────┬────┬──────┬────┐
│No.│ 회원명  │ 연락처     │ 상태 │출석│만료일│상세│
├───┼─────────┼────────────┼──────┼────┼──────┼────┤
│ 1 │홍길동   │010-1234-..│✅활성│ 45 │ D-7  │ →  │
│ 2 │김철수   │010-5678-..│⏰대기│  0 │  -   │ →  │
└───┴─────────┴────────────┴──────┴────┴──────┴────┘

데스크톱 (7열):
  1. 번호 (60px) - 파란 원형 아바타
  2. 회원명 (1fr) - 이름 + 레벨
  3. 연락처 (180px) - 전화 + 이메일
  4. 상태 (120px) - 배지
  5. 출석일수 (100px) - 숫자
  6. 만료일 (100px) - 날짜 + D-day
  7. 상세 (80px) - 화살표

모바일 (카드):
  - 번호 + 이름 + 상태 (상단)
  - 연락처 + 출석 + 만료일 (하단)
  - 전체 클릭 가능

🎨 회원 상세 페이지
------------------------------------------------------------------

1. 헤더 영역
   ┌─────────────────────────────────────┐
   │ [홍] 홍길동  ✅활성  🥋1단  ⏰D-7   │
   │      [✅출석체크] [🎫구매] [상태▼]  │
   └─────────────────────────────────────┘
   - 80px 프로필 아바타
   - 이름 (32px, 800 weight)
   - 상태 배지들
   - 빠른 액션 버튼

2. KPI 카드 (4개)
   ┌────┐ ┌────┐ ┌────┐ ┌────┐
   │ 📊 │ │ 📅 │ │ 💰 │ │ ⭐ │
   │ 45 │ │ 12 │ │ 50 │ │1200│
   │출석│ │최근│ │총액│ │포인│
   └────┘ └────┘ └────┘ └────┘

3. 탭 메뉴
   [📋 기본정보] [📊 출석현황] [🎫 회원권] [📝 메모]
   
4. 기본정보 탭
   - 그리드 레이아웃 (3열)
   - 모든 회원 정보 표시
   - 특이사항 별도 섹션

5. 출석현황 탭
   - 최근 20건 표시
   - 날짜 + 시간 + 상태
   - 배지로 시각화

6. 회원권 탭
   - 구매 이력 카드 형식
   - 활성/종료 배지
   - 시작일/종료일/금액

7. 메모 탭
   - 새 메모 작성 영역
   - 기존 메모 목록
   - 작성자 + 날짜

✨ 추가된 기능
------------------------------------------------------------------

1. 정렬 기능
   - 📅 최신순: 가입일 기준
   - 🔤 이름순: 가나다순
   - 📊 출석순: 출석일수 많은 순

2. 통합 검색
   - 이름 검색
   - 전화번호 검색
   - 이메일 검색
   - 실시간 필터링

3. 상태 필터
   - 📊 전체 상태
   - ✅ 활성만
   - ⏰ 승인대기만
   - ⏸️ 정지만
   - ❌ 해지만

4. 실시간 통계
   - 전체 X명
   - 활성 X명 (녹색)
   - 승인대기 X명 (노란색)
   - 만료임박 X명 (빨간색)

5. 출석 체크 기능
   - 원클릭 출석 체크
   - 실시간 출석 기록
   - 자동 통계 업데이트

6. 회원권 구매 모달
   - 상품 목록 표시
   - 선택 + 결제
   - 자동 만료일 설정
   - 구매 이력 기록

7. 메모 작성
   - 텍스트 입력
   - 작성자 자동 기록
   - 시간 자동 기록

8. 상태 변경
   - 드롭다운 선택
   - 원클릭 변경
   - 즉시 반영

📱 반응형 디자인
------------------------------------------------------------------

데스크톱 (>1024px):
  - 테이블 형식 (7열)
  - 가로 스크롤 없음
  - 모든 정보 한눈에

태블릿 (768-1024px):
  - 카드 형식으로 자동 전환
  - 세로 스크롤
  - 터치 최적화

모바일 (≤768px):
  - 카드 형식
  - 큰 터치 영역
  - 정보 압축 표시
  - 작은 폰트 자동 조정

🎨 디자인 상세
------------------------------------------------------------------

번호 아바타:
  - 크기: 40px × 40px
  - 원형 (border-radius: 50%)
  - 그라데이션 배경 (파란색)
  - 번호 표시 (흰색, 700 weight)
  - 그림자 효과

상태 배지:
  ✅ 활성: 녹색 배경, 녹색 텍스트
  ⏰ 대기: 노란 배경, 주황 텍스트
  ⏸️ 정지: 빨강 배경, 빨강 텍스트
  ❌ 해지: 회색 배경, 회색 텍스트

호버 효과:
  - 배경색 변경 (bg2)
  - 오른쪽 이동 (4px)
  - 부드러운 트랜지션 (0.2s)

KPI 카드:
  - 중앙 정렬
  - 큰 숫자 (28px, 800 weight)
  - 아이콘 (32px)
  - 설명 (13px, uppercase)

탭 메뉴:
  - 하단 3px 선
  - 활성: 파란색
  - 비활성: 투명
  - 호버: 부드러운 트랜지션

------------------------------------------------------------------

프론트엔드:
  - Next.js 14
  - React Hooks (useState, useEffect, useMemo)

최적화:
  - useMemo로 필터링/정렬 캐싱

  - 조건부 렌더링

🚀 사용 방법
------------------------------------------------------------------

1. 회원 관리 페이지
   http://localhost:3000/members
   
   - 검색: 이름/전화번호/이메일 입력
   - 필터: 상태 드롭다운 선택
   - 정렬: 정렬 기준 선택
   - 클릭: 회원 상세로 이동

2. 회원 상세 페이지
   http://localhost:3000/members/[id]
   
   - 출석: 출석 체크 버튼 클릭
   - 구매: 회원권 구매 버튼 → 상품 선택 → 결제
   - 상태: 상태 변경 드롭다운 → 선택
   - 메모: 메모 탭 → 입력 → 추가
