📱 테스트 방법
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. APK 빌드 완료 대기
   ⏳ 백그라운드에서 빌드 중...
   ⏳ 약 1-2분 소요

2. APK 설치
   📍 위치: pumpy-app/android/app/build/outputs/apk/release/app-release.apk
   💿 에뮬레이터 또는 실제 기기에 설치

3. 프로필 화면 테스트
   📱 프로필 탭으로 이동
   👉 프리미엄 박스 크기 확인 (30% 작아짐!)

4. 체육관 정보 확인
   👉 "내 체육관 등록 정보" 카드 클릭
   📋 회원 정보 확인 (이름, 번호, 전화, 이메일, 상태)
   📋 현재 회원권 확인 (있는 경우)
   📋 체육관 정보 확인:
      - 체육관명: Pumpy Fitness
      - 주소: 서울특별시 강남구 테헤란로 123
      - 전화번호: 02-1234-5678
      - 영업시간: 06:00 - 23:00

5. 회원권 구매 테스트
   💳 "회원권 구매하기" 버튼 클릭
   📋 회원권 옵션 확인:
      ┌──────────────────────┐
      │ 1개월 회원권 [퍼플]  │
      │ 100,000원            │
      │ [선택하기]           │
      └──────────────────────┘
      
      ┌──────────────────────┐
      │ 3개월 회원권 [핑크]  │
      │ 270,000원 [10% 할인] │
      │ [선택하기]           │
      └──────────────────────┘
   
   👉 "선택하기" 버튼 클릭
   ✅ Alert 메시지 확인
   ✅ "구매하기" 클릭 시 결제 안내 확인
