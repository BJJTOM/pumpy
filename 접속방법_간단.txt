==========================================
COCO 체육관 관리 시스템
접속 URL 확인 방법
==========================================

현재 상태:
✅ 백엔드 서버 실행 중 (Django, 8000번 포트)
✅ 프론트엔드 서버 실행 중 (Next.js, 3000번 포트)
✅ LocalTunnel 터널 생성 중

==========================================
URL 확인 방법
==========================================

화면에 2개의 CMD 창이 열려있습니다:

1번 창 (초록색): "COCO Backend Tunnel"
   └─ "your url is: https://xxxxx.loca.lt" 찾기
   └─ 이 URL을 복사하세요!

2번 창 (파란색): "COCO Frontend Tunnel"  
   └─ "your url is: https://yyyyy.loca.lt" 찾기
   └─ 💎 이 URL로 접속하세요!

==========================================
빠른 접속 방법
==========================================

방법 1: 로컬에서 확인 (가장 빠름!)
   http://localhost:3000
   ↑ 이 URL을 브라우저에 입력하세요!

방법 2: 인터넷 URL 사용
   1. CMD 창에서 URL 확인
   2. 파란색 창의 URL 복사
   3. 브라우저에 입력

==========================================
보안 화면이 나오면
==========================================

"Tunnel Password" 입력 화면이 나오면:

입력창에 아무거나 입력:
   1234

또는:
   test

그리고 "Click to Submit" 버튼 클릭!

==========================================
문제 해결
==========================================

Q: URL이 안 보여요!
A: CMD 창을 위로 스크롤하세요
   "your url is:" 다음에 URL이 있습니다

Q: 터널이 안 열렸어요!
A: 수동으로 실행:
   1. PowerShell 열기
   2. lt --port 8000 (백엔드)
   3. 새 PowerShell 창 열기  
   4. lt --port 3000 (프론트엔드)

Q: 접속이 안 돼요!
A: 먼저 로컬에서 확인:
   http://localhost:3000

==========================================
로컬 접속 (추천!)
==========================================

가장 빠르고 안정적인 방법:

http://localhost:3000

브라우저에서 위 주소로 접속하시면
바로 COCO 시스템을 사용할 수 있습니다!

페이지:
- 대시보드: http://localhost:3000
- 회원 관리: http://localhost:3000/members
- 회원 가입: http://localhost:3000/signup
- 회원 앱: http://localhost:3000/app
