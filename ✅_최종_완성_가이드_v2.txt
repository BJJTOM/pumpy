방법 1: Expo Go (즉시 테스트) ⭐
   1. Play Store에서 "Expo Go" 설치
   2. 컴퓨터와 같은 Wi-Fi 연결
   3. npx expo start 실행
   4. QR 코드 스캔
   5. 서버 URL 입력:
      http://YOUR_IP:3000
      (예: http://172.30.1.44:3000)
   6. 완료!

방법 2: APK 빌드 (배포용)
   1. EAS CLI 설치:
      npm install -g eas-cli
   
   2. Expo 로그인:
      eas login
   
   3. 프로젝트 설정:
      cd pumpy-mobile
      eas build:configure
   
   4. APK 빌드:
      eas build --platform android --profile preview
   
   5. 빌드 완료 후 다운로드
      (약 15~20분 소요)

ngrok 사용 (1주일 무료 테스트):

1. ngrok 설치

   - Windows용 다운로드
   - ngrok config add-authtoken YOUR_TOKEN

2. 터널 실행
   터미널 1: python manage.py runserver 0.0.0.0:8000
   터미널 2: npm run dev (웹)
   터미널 3: ngrok http 3000
   터미널 4: ngrok http 8000

3. ngrok URL 확인
   예: https://abc123.ngrok-free.app

4. 앱에서 사용
   - 앱 실행
   - 서버 설정
   - ngrok URL 입력
   - 저장

5. 완료!

Q: 웹이 작동하지 않습니다
A: 1. 백엔드가 실행 중인지 확인
   2. http://localhost:8000/api/members/ 접속 테스트
   3. 브라우저 콘솔에서 오류 확인

Q: 앱에서 "Connection refused"
A: 1. 서버 URL이 올바른지 확인
   2. 컴퓨터와 스마트폰이 같은 Wi-Fi인지 확인
   3. PC 방화벽에서 8000번 포트 허용 확인
   4. ipconfig로 IP 주소 재확인

Q: Expo Go가 QR을 인식하지 못합니다
A: 1. 같은 Wi-Fi 확인
   2. Expo 터미널에서 'a' 키로 에뮬레이터 사용
   3. 또는 터널 모드: npx expo start --tunnel

Q: npm install 오류
A: 1. Node.js 버전 확인 (18+ 필요)
   2. node_modules 삭제 후 재설치:
      rmdir /s /q node_modules
      npm install

Q: EAS Build 오류
A: 1. Expo 계정 확인
   2. app.json에서 projectId 확인
   3. eas.json 설정 확인
