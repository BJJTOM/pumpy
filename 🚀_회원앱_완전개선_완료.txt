🎨 신체 정보 수정 기능
------------------------------------------------------------------

수정 모드:
  1. [✏️수정] 버튼 클릭
  2. 각 항목에 입력 필드 표시
  3. 숫자 입력 가능
  4. [✅저장] 버튼으로 저장
  5. [취소] 버튼으로 취소

자동 계산:
  - BMI = 체중(kg) / (키(m))²
  - 상태 분류:
    * < 18.5: 저체중 (녹색)
    * 18.5-23: 정상 (파란색)
    * 23-25: 과체중 (주황색)
    * >= 25: 비만 (빨간색)

저장 위치:
  - localStorage에 저장
  - 키: `body_stats_${user.id}`
  - 형식: JSON { weight, muscle, fat, height }

📷 사진 업로드 시스템
------------------------------------------------------------------

업로드 방법:
  1. 프로필 사진 영역 클릭
  2. 파일 선택 대화상자
  3. 이미지 파일 선택
  4. 자동 업로드 및 표시

파일 제한:
  - 허용 형식: image/*
  - 최대 크기: 5MB
  - 초과 시 경고 메시지

저장 방식:
  - Base64 인코딩
  - localStorage 저장
  - 키: `user_photo_${user.id}`
  - 페이지 새로고침 시에도 유지

AI 효과:
  - 반투명 그라데이션 오버레이
  - mix-blend-mode: overlay
  - 색상: 파란색-보라색 그라데이션
  - AI 캐릭터처럼 보이는 효과

적용 위치:
  - 메인 페이지 프로필 사진
  - 메인 페이지 AI 캐릭터 카드
  - 프로필 페이지 프로필 사진

📊 데이터 저장 구조
------------------------------------------------------------------

localStorage 키:
  1. user_photo_${userId}
     - 사진 Base64 문자열
     - 예: "data:image/jpeg;base64,..."

  2. body_stats_${userId}
     - JSON 문자열
     - 예: {"weight":70,"muscle":35,"fat":15,"height":175}

읽기:
  const photo = localStorage.getItem(`user_photo_${userId}`)
  const stats = JSON.parse(localStorage.getItem(`body_stats_${userId}`))

쓰기:
  localStorage.setItem(`user_photo_${userId}`, base64)
  localStorage.setItem(`body_stats_${userId}`, JSON.stringify(stats))

------------------------------------------------------------------

시나리오 1: 첫 사용자
  1. 앱 홈 접속
  2. "나만의 AI 캐릭터" 카드 표시 (사진 없음)
  3. 카드 클릭 → 프로필 페이지 이동
  4. 프로필 사진 클릭 → 파일 선택
  5. 사진 업로드 → AI 캐릭터 생성
  6. 신체 정보 수정
  7. 홈으로 돌아가면 모든 정보 표시

시나리오 2: 기존 사용자
  1. 앱 홈 접속
  2. 저장된 사진 자동 로드
  3. 신체 정보 자동 표시
  4. AI 캐릭터 카드에 사진 표시
  5. 필요 시 프로필에서 수정

시나리오 3: 신체 정보 업데이트
  1. 프로필 페이지 접속
  2. [✏️수정] 버튼 클릭
  3. 체중, 근육량, 체지방률 입력
  4. [✅저장] 클릭
  5. BMI 자동 재계산
  6. 홈 화면에 즉시 반영
