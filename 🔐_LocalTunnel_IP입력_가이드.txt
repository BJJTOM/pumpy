==========================================
COCO - LocalTunnel IP 입력 가이드
==========================================

LocalTunnel 접속 시 IP 주소를 요구하는 보안 화면이 나타날 수 있습니다.
아래 방법으로 간단히 해결할 수 있습니다!

==========================================
방법 1: 본인 IP 주소 확인 및 입력
==========================================

1. 웹 브라우저에서 아래 사이트 접속:
   https://www.whatismyip.com/
   또는
   https://ipconfig.me/
   또는
   네이버 검색: "내 IP 주소"

2. 나온 IP 주소를 복사합니다
   (예: 123.456.789.012)

3. LocalTunnel 보안 화면의 "Tunnel Password" 입력창에 붙여넣기

4. "Click to Submit" 버튼 클릭

5. 접속 완료! ✅

==========================================
방법 2: 개발자 옵션 사용 (더 빠름!)
==========================================

1. LocalTunnel 보안 화면 아래로 스크롤

2. "Are you the developer?" 섹션 찾기

3. 링크 클릭 또는 "Continue" 버튼 클릭

4. 바로 접속 완료! ✅

==========================================
방법 3: 다른 브라우저 시도
==========================================

Chrome, Edge, Firefox 등 다른 브라우저로 시도하면
보안 화면이 다르게 나타날 수 있습니다.

==========================================
자주 묻는 질문
==========================================

Q: IP 주소를 입력했는데 "invalid"라고 나와요
A: 
   1. IP 주소가 올바른지 다시 확인
   2. whatismyip.com에서 다시 확인
   3. 방법 2 (개발자 옵션) 시도

Q: 보안 화면이 계속 나와요
A:
   1. 브라우저 쿠키 허용 확인
   2. 시크릿 모드로 접속 시도
   3. 다른 브라우저로 시도

Q: 모바일에서도 IP 입력이 필요한가요?
A:
   네, 동일한 방법으로 진행하면 됩니다.
   모바일 브라우저에서:
   1. whatismyip.com 접속
   2. IP 복사
   3. LocalTunnel 화면에 붙여넣기

==========================================
핵심 요약
==========================================

1. whatismyip.com → IP 복사
2. LocalTunnel 화면 → IP 붙여넣기
3. Submit 클릭
4. 완료! 🎉

또는

1. LocalTunnel 화면 아래로 스크롤
2. "Are you the developer?" 클릭
3. 완료! 🎉

==========================================
도움말
==========================================

- LocalTunnel은 무료 서비스이며 보안을 위해 IP 확인을 요구합니다
- 한 번 입력하면 같은 브라우저에서는 다시 입력하지 않아도 됩니다
- IP 주소는 인터넷 공유기가 바뀌면 달라질 수 있습니다
