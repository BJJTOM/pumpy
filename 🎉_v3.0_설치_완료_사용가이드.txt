  🚀 앱 실행 방법
═══════════════════════════════════════════════════════

[방법 1] 에뮬레이터 앱 목록에서:
  1. 에뮬레이터 하단의 앱 목록 버튼 클릭
  2. "펌피" 또는 "Pumpy" 앱 아이콘 찾기
  3. 클릭하여 실행

[방법 2] 명령어로 실행:
  & "$env:LOCALAPPDATA\Android\Sdk\platform-tools\adb.exe" shell monkey -p com.pumpy.gym -c android.intent.category.LAUNCHER 1

[방법 3] Metro Bundler와 함께:
  1. pumpy-app 폴더에서 Metro 시작: npm start
  2. 에뮬레이터에서 앱 실행
  3. 자동으로 Metro에 연결됨
