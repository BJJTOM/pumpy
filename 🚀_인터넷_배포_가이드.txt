🌐 COCO - 인터넷 배포 가이드
(LocalTunnel IP 문제 해결)
==========================================

LocalTunnel이 IP를 계속 요구하는 문제가 있네요!
더 쉬운 방법으로 바꿔드리겠습니다.

==========================================
✅ 해결 방법: Ngrok 사용
==========================================

Ngrok은 IP 제한이 없고 무료로 사용 가능합니다!

==========================================
📥 설치 방법
==========================================

방법 1: 직접 다운로드 (추천!)
   1. https://ngrok.com/download 접속
   2. Windows 버전 다운로드
   3. 압축 해제
   4. ngrok.exe를 C:\Users\guddn\Downloads\COCO\ 에 복사

방법 2: 자동 설치
   PowerShell 관리자 권한으로 실행:
   
   choco install ngrok
   
   (Chocolatey 필요)

==========================================
🚀 사용 방법
==========================================

1단계: PowerShell 2개 열기

PowerShell 1 (백엔드):
   cd C:\Users\guddn\Downloads\COCO
   .\ngrok.exe http 8000

PowerShell 2 (프론트엔드):
   cd C:\Users\guddn\Downloads\COCO
   .\ngrok.exe http 3000

2단계: URL 확인
   각 창에 이렇게 나옵니다:
   
   Forwarding  https://xxxx-xxx-xxx-xxx.ngrok-free.app -> http://localhost:8000
   
   이 https:// URL을 복사하세요!

3단계: 프론트엔드 재시작
   프론트엔드 PowerShell 2의 URL을 확인한 후
   브라우저에서 접속하면 됩니다!

==========================================

==========================================
🆘 문제 해결
==========================================

Q: ngrok.exe가 실행이 안 돼요!
A: Windows Defender가 차단할 수 있습니다
   "자세한 정보" → "실행" 클릭

Q: URL이 안 보여요!
A: 몇 초 기다리면 화면에 나타납니다
   "Forwarding" 줄의 https:// URL 사용

Q: 접속이 안 돼요!
A: 
   1. 백엔드(8000), 프론트엔드(3000) 서버 실행 확인
   2. ngrok이 올바른 포트로 실행 중인지 확인
   3. 방화벽 설정 확인
