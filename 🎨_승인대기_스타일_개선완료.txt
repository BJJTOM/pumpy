✨ 개선된 디자인 요소
------------------------------------------------------------------

1. 헤더 디자인
   - 큰 이모지 아이콘 (⏰)
   - 더 큰 타이틀 (32px, 800 weight)
   - 동적 설명 텍스트 (회원 수 강조)
   - 하단 구분선으로 섹션 구분

2. 회원 카드 디자인
   - 2단 레이아웃 (헤더 + 정보 영역)
   - 그라데이션 배경 적용
   - 프로필 아바타 추가 (이름 첫 글자)
   - 2px 테두리로 강조
   - 호버 효과

3. 버튼 스타일
   - 승인: 녹색 그라데이션 + 그림자
   - 거부: 회색 배경 + 빨간 텍스트
   - 아이콘 + 텍스트 조합
   - 더 큰 패딩 (12px 24px)

4. 정보 표시
   - 그리드 레이아웃 (3열)
   - 라벨 대문자 + 작은 글씨
   - 값 큰 글씨 + 굵게
   - 이모지 아이콘 추가

5. 메모 섹션
   - 그라데이션 배경
   - 테두리 + 둥근 모서리
   - 제목 + 본문 구분
   - pre-wrap으로 줄바꿈 유지

6. 빈 상태 디자인
   - 큰 이모지 (64px)
   - 그라데이션 배경
   - 명확한 메시지
   - 중앙 정렬

7. 로딩 스피너
   - 회전 애니메이션
   - 브랜드 컬러 적용
   - 중앙 정렬
   - 설명 텍스트

🎨 디자인 상세
------------------------------------------------------------------

헤더 영역:
  - 이모지 크기: 32px
  - 타이틀: 32px, 800 weight
  - 설명: 15px, 회원 수 강조 (파란색)
  - 하단 구분선: 2px solid

회원 카드:
  ┌─────────────────────────────────────┐
  │ [아바타] 홍길동  [승인] [거부]      │ ← 헤더 (그라데이션)
  │         ⏰ 승인 대기중               │
  ├─────────────────────────────────────┤
  │ 📱 전화번호    📧 이메일    📅 신청일│
  │ 010-1234-5678  hong@...   2025.10.15│
  │                                      │
  │ 📝 신청 메모                         │
  │ [메모 내용]                          │
  └─────────────────────────────────────┘

아바타 디자인:
  - 크기: 48px × 48px
  - 원형 (border-radius: 50%)
  - 그라데이션 배경 (파란색)
  - 이름 첫 글자 표시
  - 그림자 효과

버튼 디자인:
  승인 버튼:
    - 배경: 녹색 그라데이션
    - 그림자: 0 4px 12px rgba(0, 217, 165, 0.3)
    - 텍스트: 흰색, 700 weight
    - 아이콘: ✅
  
  거부 버튼:
    - 배경: 회색
    - 테두리: 1px solid
    - 텍스트: 빨간색, 700 weight
    - 아이콘: ❌

📱 반응형 디자인
------------------------------------------------------------------

데스크톱 (>768px):
  - 정보 그리드: 3열
  - 버튼: 가로 배치
  - 최대 너비: 1200px

모바일 (≤768px):
  - 정보 그리드: 1열 (자동 조정)
  - 버튼: 세로 배치 (flexWrap)
  - 전체 너비 사용
  - 패딩 축소

🎯 사용된 CSS 기법
------------------------------------------------------------------

1. 그라데이션 배경
   background: linear-gradient(135deg, color1 0%, color2 100%)

2. 그림자 효과
   box-shadow: 0 4px 12px rgba(49, 130, 246, 0.3)

3. 그리드 레이아웃
   display: grid
   grid-template-columns: repeat(auto-fit, minmax(250px, 1fr))

4. 플렉스 박스
   display: flex
   justify-content: space-between
   align-items: center

5. 트랜지션
   transition: all 0.2s ease

6. 애니메이션
   @keyframes spin { ... }
   animation: spin 0.8s linear infinite
