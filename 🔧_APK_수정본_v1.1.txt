1️⃣ 기존 앱 삭제 (있다면)
   - 설정 → 앱 → 펌피 → 삭제

2️⃣ 새 APK 설치
   - Pumpy_v1.1_Fixed.apk 파일을 폰으로 전송
   - 파일 클릭하여 설치

3️⃣ 서버 IP 주소 확인
   컴퓨터에서:
   - 명령 프롬프트(CMD) 실행
   - ipconfig 입력
   - IPv4 주소 확인 (예: 172.30.1.44)

4️⃣ 서버 실행
   백엔드 서버:
   cd C:\Users\guddn\Downloads\COCO\gym_api
   .\.venv\Scripts\python manage.py runserver 0.0.0.0:8000

   프론트엔드 서버:
   cd C:\Users\guddn\Downloads\COCO\gym_web
   npm run dev

5️⃣ 앱 실행 및 설정
   - 앱 실행
   - 자동으로 "서버 설정" 화면이 표시됨
   - 서버 URL 입력: http://172.30.1.44:3000
     (172.30.1.44를 실제 IP로 변경)
   - "저장 및 연결 테스트" 버튼 클릭
   - 연결 성공 시 메인 화면으로 자동 이동

Q: 여전히 흰 화면이 보여요
A: 1. 앱을 완전히 종료 (백그라운드에서도)
   2. 앱 데이터 삭제: 설정 → 앱 → 펌피 → 저장공간 → 데이터 삭제
   3. 앱 재실행
   4. 서버 URL 다시 입력

Q: "서버에 연결할 수 없습니다" 오류
A: 1. 서버가 실행 중인지 확인
   2. 브라우저에서 http://입력한IP:3000 접속 테스트
   3. 방화벽 설정 확인
   4. 같은 Wi-Fi 연결 확인

Q: IP 주소가 계속 바뀌어요
A: 1. 공유기 설정에서 고정 IP 할당
   2. 또는 매번 앱에서 새로운 IP 입력
   3. 서버 설정 페이지에서 언제든 변경 가능

Q: 서버 설정 페이지로 다시 가고 싶어요
A: 1. 앱 설정 메뉴 (아직 구현 안 됨)
   2. 또는 앱 데이터 삭제 후 재설정
