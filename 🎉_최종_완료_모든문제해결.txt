🔧 서버 재시작 방법 (필요시)
------------------------------------------------------------------
1. 모든 프로세스 종료:
   PowerShell에서 실행:
   Stop-Process -Name "node","python" -Force

2. 백엔드 시작:
   cd gym_api
   .venv\Scripts\python manage.py runserver 0.0.0.0:8000

3. 프론트엔드 시작:
   cd gym_web
   $env:NEXT_PUBLIC_API_BASE="http://localhost:8000/api"
   npm run dev

또는 START.bat 파일 실행!
