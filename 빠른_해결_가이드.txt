✅ 단계별 진행:

1. 백엔드 서버 실행 ✅ (완료됨!)
   └─ 새 PowerShell 창이 열려 있습니다

2. 앱 실행
   └─ APK 설치 후 앱 실행

3. 서버 URL 입력
   └─ "서버 설정" 화면에서 위 URL 입력

4. 연결 테스트
   └─ "연결 테스트 & 저장" 버튼 클릭

5. 성공! 🎉
   └─ 로그인 화면으로 자동 이동


⚠️  중요 확인 사항:

□ PC와 안드로이드 기기가 같은 Wi-Fi에 연결되어 있나요?
□ 백엔드 서버 창을 닫지 않았나요?
□ URL을 정확히 입력했나요? (특히 끝에 /api)

💡 문제 해결:

❌ "서버에 연결할 수 없습니다"
   → Wi-Fi 확인, 서버 실행 확인

❌ "타임아웃"
   → 네트워크 신호 확인, 라우터 재시작

❌ "404 Not Found"
   → URL 끝에 /api 확인
