==========================================
🌐 COCO 체육관 - 인터넷 접속 URL
==========================================

✅ LocalTunnel 터널이 생성되었습니다!

지금 화면에 2개의 PowerShell 창이 열려있습니다:

==========================================
📋 URL 확인 방법
==========================================

1️⃣ 초록색 창: "COCO Backend Tunnel"
   - 위로 스크롤하세요
   - "your url is: https://xxxxx.loca.lt" 찾기
   - 📝 이 URL을 메모하세요 (백엔드 URL)

2️⃣ 파란색 창: "COCO Frontend Tunnel"
   - 위로 스크롤하세요
   - "your url is: https://yyyyy.loca.lt" 찾기
   - 🌟 이 URL이 접속 주소입니다!

==========================================
🔓 보안 화면 통과 방법
==========================================

URL에 접속하면 "Tunnel Password" 화면이 나옵니다.

방법 1: IP 입력 건너뛰기
   1. 화면 아래 "Are you the developer?" 찾기
   2. 링크를 클릭하거나
   3. "Click to Continue" 버튼 찾아서 클릭

방법 2: 현재 IP 확인
   아래 사이트에서 본인 IP 확인:
   https://www.whatismyip.com/
   
   확인한 IP를 입력창에 입력
   예: 123.456.789.012

==========================================
📱 핸드폰/다른 컴퓨터에서 접속
==========================================

파란색 창의 URL을 다른 기기에서 열면:
1. 보안 화면이 나옵니다
2. IP 주소를 입력하거나
3. "Are you the developer?" 클릭
4. COCO 시스템 접속 완료! 🎉

==========================================
🛠️ 문제 해결
==========================================

Q: URL이 안 보여요!
A: PowerShell 창을 위로 스크롤하세요
   "your url is:" 다음에 긴 URL이 있습니다

Q: 보안 화면을 못 통과하겠어요!
A: 
   방법 1) 화면 아래로 스크롤해서 링크 찾기
   방법 2) www.whatismyip.com 에서 IP 확인 후 입력
   방법 3) 다른 브라우저로 시도 (크롬, 엣지 등)

Q: 터널이 끊겼어요!
A: PowerShell 창에서 Ctrl+C 누르고
   다시 명령 실행:
   
   백엔드: lt --port 8000
   프론트엔드: lt --port 3000

==========================================
💡 주의사항
==========================================

1. 터널은 무료로 제공되며 일정 시간 후 끊길 수 있습니다
2. 끊기면 새로운 URL이 생성됩니다
3. 중요한 데이터는 로컬에 백업하세요
4. 보안을 위해 테스트 목적으로만 사용하세요

==========================================
🚀 빠른 접속
==========================================

1. 파란색 PowerShell 창 열기
2. 위로 스크롤
3. "your url is: https://xxxxx.loca.lt" 복사
4. 브라우저에 붙여넣기
5. 보안 화면 통과
6. 접속 완료! 🎉
