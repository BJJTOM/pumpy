설치 방법 (1분):

1️⃣ 서버 IP 확인:
   컴퓨터 CMD → ipconfig
   → IPv4 주소: 예) 172.30.1.44

2️⃣ 스마트폰에서:
   Chrome 브라우저 열기
   → http://172.30.1.44:3000 접속

3️⃣ 홈 화면에 추가:
   Chrome 메뉴 (⋮) 클릭
   → "홈 화면에 추가" 선택
   → "추가" 확인

4️⃣ 완료!
   홈 화면에 "펌피" 아이콘 생성
   앱처럼 실행 가능!

✅ 설치 간단 (1분)
✅ 웹과 100% 동일한 스타일
✅ 모든 기능 작동
✅ 업데이트 즉시 반영
✅ 설치 파일 불필요
✅ 빠른 성능

단점:
❌ 같은 Wi-Fi 필요 (또는 ngrok)

Progressive Web App (PWA)란?
- 웹사이트를 앱처럼 설치
- 홈 화면 아이콘
- 전체 화면 실행
- 빠른 로딩
- 오프라인 지원 가능

설치 방법 (Android):

1. Chrome에서 웹사이트 접속
2. 메뉴 (⋮) → "홈 화면에 추가"
3. 이름 확인: "펌피"
4. "추가" 클릭
5. 홈 화면에 아이콘 생성

설치 방법 (iOS/iPhone):

1. Safari에서 웹사이트 접속
2. 공유 버튼 (⬆️) 클릭
3. "홈 화면에 추가" 선택
4. "추가" 확인

접속 방법:
✅ PC: http://localhost:3000
✅ 모바일 (같은 Wi-Fi): http://YOUR_IP:3000
✅ 모바일 (외부): ngrok URL

기능:
✅ 모든 기능 정상 작동
✅ 원래 스타일 유지 (Toss UI)
✅ 완벽한 반응형
