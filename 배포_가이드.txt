========================================
  🚀 Vercel 배포 가이드 (가장 빠른 방법)
========================================

【방법 1】 Vercel 웹사이트에서 직접 배포 (추천!)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. https://vercel.com 접속
2. GitHub 계정으로 로그인
3. "Add New" → "Project" 클릭
4. "Import Git Repository" → "Continue with GitHub"
5. "BJJTOM" 계정 선택
6. "Import Project" 페이지에서:
   - Repository 권한 허용
   - 저장소 선택 (또는 새로 만들기)

7. 프로젝트 설정:
   - Framework Preset: Next.js
   - Root Directory: ./gym_web
   - Build Command: npm run build
   - Output Directory: .next
   - Install Command: npm install

8. "Deploy" 클릭!

→ 약 2-3분 후 배포 완료!
→ URL이 자동으로 생성됨 (예: pumpy-gym.vercel.app)


【방법 2】 GitHub Desktop 사용 (조금 더 빠름)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. GitHub Desktop 다운로드 & 설치
   https://desktop.github.com/

2. GitHub Desktop 실행
   - BJJTOM 계정으로 로그인

3. "Add" → "Add Existing Repository"
   - C:\Users\guddn\Downloads\COCO 선택

4. "Publish repository" 클릭
   - Name: pumpy-gym
   - Private 선택 (무료)
   - "Publish Repository"

5. https://vercel.com 접속
   - 위의 방법 1의 3번부터 진행


【방법 3】 Vercel CLI로 배포 (터미널)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PowerShell에서:

1. cd C:\Users\guddn\Downloads\COCO\gym_web

2. vercel login
   → 브라우저가 열림
   → GitHub 계정으로 로그인

3. vercel
   → 질문에 답변:
   - Set up and deploy: Y
   - Which scope: 본인 계정 선택
   - Link to existing project: N
   - Project name: pumpy-gym
   - Directory: ./
   - Override settings: N

4. 배포 완료!
