         🚨 문제 해결
═══════════════════════════════════════════════════════════════

❌ "Network Error" 발생 시:
   1. AWS 서버가 실행 중인지 확인
      → http://3.27.28.175:8000 접속 테스트
   
   2. test-login.html로 서버 연결 테스트
      → C:\Users\guddn\Downloads\COCO\test-login.html
   
   3. 휴대폰이 인터넷에 연결되어 있는지 확인
      (Wi-Fi 또는 모바일 데이터)

❌ 로그인이 안 될 때:
   1. 기존 앱 데이터를 완전히 삭제했는지 확인
   2. 이메일/비밀번호 정확히 입력했는지 확인
   3. 회원가입으로 새 계정 생성

❌ 여전히 서버 설정 화면이 나타날 때:
   1. 브라우저 캐시 완전 삭제
   2. 앱 데이터 완전 삭제 후 재설치
   3. localStorage 수동 삭제:
      - 브라우저: F12 → Application → Local Storage → 모두 삭제
      - 앱: 설정 → 앱 → Pumpy → 데이터 삭제
