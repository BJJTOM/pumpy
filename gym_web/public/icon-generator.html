💪 펌피 아이콘 생성기

간단한 플레이스홀더 아이콘을 생성합니다. 실제 앱 배포 시에는 전문 디자이너가 만든 아이콘을 권장합니다.

📝 사용 방법

  1. 아래 버튼을 클릭하여 각 크기별 아이콘을 다운로드
  2. 다운로드된 파일들을 gym_web/public/icons/ 폴더에 저장
  3. 앱 빌드 및 배포
512×512 메인 아이콘
192×192 홈 화면
144×144 태블릿
72×72 작은 아이콘
