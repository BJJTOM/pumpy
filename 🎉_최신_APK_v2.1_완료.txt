   cd gym_web
   npm run dev

   - Next.js 웹: 실행 중 (3000번 포트)

Q: 앱에서 "Connection refused" 오류
A: 1. 서버 실행 확인:
      - 백엔드 (8000번 포트)
      - 웹 (3000번 포트)
   2. 서버 URL 확인
   3. 방화벽 확인

Q: 스타일이 이전 버전처럼 보입니다
A: 1. 기존 앱 완전히 삭제
   2. 스마트폰 재시작
   3. 최신 APK (v2.1) 설치

Q: 데이터가 로딩되지 않습니다
A: 1. 서버 실행 확인
   2. 서버 URL 정확한지 확인
   3. 네트워크 연결 확인

Q: 서버 URL을 모르겠습니다
A: CMD에서 ipconfig 실행
   → IPv4 주소: 예: 172.30.1.44
   → 입력: http://172.30.1.44:3000
