========================================
  🚀 사용 방법
========================================

【1단계】 서버 터널 시작
-----------------------------------------
컴퓨터에서 START_TUNNEL.bat 더블클릭

→ 2개의 터널 창이 열림 (닫지 마세요!)
  - Frontend Tunnel (포트 3000)
  - Backend Tunnel (포트 8000)


【2단계】 APK 설치 및 실행
-----------------------------------------
1. Pumpy_Final_v2.apk를 휴대폰으로 전송
2. APK 설치 (보안 설정 허용)
3. 앱 실행


【3단계】 서버 설정
-----------------------------------------
앱을 처음 실행하면 "서버 설정" 화면이 나타남

서버 URL 입력:
  https://pumpy-app-2025.loca.lt

"저장 및 연결" 버튼 클릭


【4단계】 완료!
-----------------------------------------
앱이 로드되고 모든 기능을 사용할 수 있습니다!

========================================
  📱 APK 작동 원리
========================================

1. APK는 iframe을 통해 웹 서버에 접속
2. 모든 UI와 기능은 Next.js 앱에서 제공
3. API 데이터는 Django 백엔드에서 가져옴
4. LocalStorage에 서버 URL을 저장하여
   한 번만 설정하면 계속 사용 가능

┌─────────────┐    iframe     ┌─────────────┐
│   APK 앱    │──────────────→│ Next.js 앱  │
│ (Capacitor) │               │ (포트 3000) │
└─────────────┘               └─────────────┘
                                     │
                                     │ API 호출
                                     ↓
                              ┌─────────────┐
                              │  Django API │
                              │ (포트 8000) │
                              └─────────────┘

========================================
  ⚠️ 주의사항
========================================

1. 터널 실행 필수
   - START_TUNNEL.bat가 실행 중이어야 함
   - 2개의 창(Frontend, Backend) 모두 필요
   - 터널 창을 닫으면 연결 끊김

2. 처음 접속 시
   - LocalTunnel 보안 화면이 나올 수 있음
   - "Click to Continue" 버튼 클릭
   - 또는 외부 IP 주소 입력 (한 번만)

3. 서버 URL 변경
   - 앱 내에서 "설정" → "서버 재설정"
   - 또는 앱 데이터 삭제 후 재설정

4. 데이터 저장
   - LocalStorage에 서버 URL 저장
   - 앱을 삭제하면 다시 설정 필요

========================================
  🔧 문제 해결
========================================

Q: 앱이 "로딩 중"에서 멈춰있어요
A: 1. 터널이 실행 중인지 확인
   2. 서버 URL이 정확한지 확인
   3. 브라우저에서 먼저 접속 테스트
      https://pumpy-app-2025.loca.lt

Q: "서버에 연결할 수 없습니다" 오류
A: 1. START_TUNNEL.bat 실행 확인
   2. Frontend 서버 확인 (포트 3000)
   3. Backend 서버 확인 (포트 8000)
   4. 인터넷 연결 확인

Q: 데이터가 로드되지 않아요
A: 1. Backend API 정상 작동 확인
   2. 브라우저에서 API 테스트
      https://pumpy-api-2025.loca.lt/api/members/
   3. 앱을 완전히 종료 후 재실행

Q: LocalTunnel이 자주 끊겨요
A: 1. 무료 서비스의 제한사항
   2. 터널 재실행: START_TUNNEL.bat
   3. 필요시 다른 subdomain 사용

Q: 같은 Wi-Fi에서만 사용하고 싶어요
A: 서버 URL을 http://172.30.1.44:3000
   으로 설정하세요 (터널 불필요)
