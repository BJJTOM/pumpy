🌐 LocalTunnel 접속 방법
==========================================

1. PowerShell 2개 창 열기

2. 백엔드 터널 (초록색 창):
   cd C:\Users\guddn\Downloads\COCO
   lt --port 8000

3. 프론트엔드 터널 (파란색 창):
   cd C:\Users\guddn\Downloads\COCO
   lt --port 3000

4. 파란색 창의 URL로 접속
   예: https://xxxxx.loca.lt

5. IP 입력 화면 나오면:
   방법 1) whatismyip.com 에서 IP 확인 후 입력
   방법 2) "Are you the developer?" 클릭

6. 완료! 🎉
