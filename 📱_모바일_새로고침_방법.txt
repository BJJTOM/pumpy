📱 모바일에서 강력 새로고침 방법
==================================================================

🔄 방법 1: 브라우저 캐시 삭제 (가장 확실!)
------------------------------------------------------------------

📱 안드로이드 (Chrome):
  1. 주소창 옆 (i) 또는 🔒 아이콘 클릭
  2. "사이트 설정" 선택
  3. "저장 공간" 또는 "데이터 삭제" 클릭
  4. "데이터 삭제" 확인
  5. 페이지 새로고침

  또는:
  1. Chrome 설정 (⋮) → 설정
  2. 개인정보 보호 및 보안
  3. 인터넷 사용 기록 삭제
  4. "캐시된 이미지 및 파일" 체크
  5. 데이터 삭제

📱 iPhone (Safari):
  1. 설정 앱 열기
  2. Safari 선택
  3. "방문 기록 및 웹사이트 데이터 지우기"
  4. 확인
  5. Safari로 돌아가서 페이지 새로고침

  또는:
  1. 주소창 길게 누르기
  2. "새로고침" 옵션 선택

==================================================================

🔄 방법 2: 시크릿 모드 (빠른 확인!)
------------------------------------------------------------------

📱 안드로이드 (Chrome):
  1. Chrome 메뉴 (⋮) → 새 시크릿 탭
  2. http://172.30.1.44:3000 입력

📱 iPhone (Safari):
  1. Safari 하단 탭 버튼 길게 누르기
  2. "새로운 비공개 탭" 선택
  3. http://172.30.1.44:3000 입력

==================================================================

🔄 방법 3: 브라우저 완전 종료 후 재시작
------------------------------------------------------------------

📱 안드로이드:
  1. 최근 앱 버튼 (□) 누르기
  2. Chrome 위로 스와이프하여 종료
  3. Chrome 다시 실행
  4. http://172.30.1.44:3000 접속

📱 iPhone:
  1. 홈 버튼 2번 클릭 또는 화면 아래에서 위로 스와이프
  2. Safari 위로 스와이프하여 종료
  3. Safari 다시 실행
  4. http://172.30.1.44:3000 접속

==================================================================

🔄 방법 4: 개발자 도구 사용 (고급)
------------------------------------------------------------------

📱 안드로이드 (Chrome):
  1. 주소창에 chrome://inspect 입력
  2. "Discover network targets" 활성화
  3. 현재 탭 선택
  4. "Reload" 버튼 길게 누르기
  5. "Empty Cache and Hard Reload" 선택

==================================================================

✅ 가장 간단한 방법 (추천!)
------------------------------------------------------------------

1. Chrome 또는 Safari 완전 종료
2. 앱 다시 실행
3. http://172.30.1.44:3000 입력
4. 잠시 기다리면 데이터 표시!

또는

1. 시크릿/비공개 모드로 접속
   → 캐시 없이 새로 로드됨

💡 여전히 "로딩 중..." 상태라면?
------------------------------------------------------------------
1. WiFi가 같은 네트워크인지 확인
   - PC와 모바일이 같은 WiFi에 연결되어 있어야 함

2. IP 주소 확인
   - PC에서 PowerShell 실행:
     ipconfig
   - IPv4 주소 확인 (예: 172.30.1.44)
   - 모바일에서 해당 IP로 접속

3. 방화벽 확인
   - Windows 방화벽에서 8000, 3000 포트 허용

4. 백엔드 실행 확인
   - PC 브라우저에서 http://localhost:8000/admin/
   - 관리자 페이지가 열리면 백엔드 정상

==================================================================

2. 브라우저 종류 (Chrome/Safari/기타)
3. 어떤 에러 메시지가 보이는지
4. PC에서는 정상 작동하는지
