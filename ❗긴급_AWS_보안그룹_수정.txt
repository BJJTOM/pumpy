🚨 문제 확인됨:
- Connection timed out (작업 시간 초과)
- AWS 보안 그룹에서 모든 포트가 차단되어 있습니다!

═══════════════════════════════════════════════════════════

📱 지금 바로 실행하세요! (3분)

1. 브라우저에서 새 탭을 엽니다

2. 다음 주소로 이동:
   👉 https://console.aws.amazon.com/ec2/

3. 로그인

4. 왼쪽 메뉴에서 "Instances" 클릭

5. "pumpy-server" 또는 "i-065a56c91b35d3fee" 선택
   (체크박스 클릭)

6. 화면 아래쪽 "Security" 탭 클릭

7. "Security groups:" 옆에 있는 파란색 링크 클릭
   (sg-xxxxxxxxx 형식)

8. "Inbound rules" 탭 선택

9. "Edit inbound rules" 버튼 클릭 (주황색)

10. 다음 규칙들을 추가:

    【Add rule 버튼 클릭】
    Type: HTTP
    Source: Anywhere-IPv4 (0.0.0.0/0)

    【Add rule 버튼 클릭】
    Type: HTTPS
    Source: Anywhere-IPv4 (0.0.0.0/0)

    【Add rule 버튼 클릭】
    Type: Custom TCP
    Port range: 8000
    Source: Anywhere-IPv4 (0.0.0.0/0)

    【SSH가 없다면 추가】
    Type: SSH
    Source: Anywhere-IPv4 (0.0.0.0/0)

11. "Save rules" 버튼 클릭 (주황색)

12. 완료!
