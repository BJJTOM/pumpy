방법: ngrok 사용 (가장 쉽고 안정적)

1. ngrok 설치 (1분)
   - https://ngrok.com 회원가입
   - Windows용 다운로드
   - Authtoken 설정:
     ngrok config add-authtoken YOUR_TOKEN

2. 서버 실행 (2분)
   터미널 1: cd gym_api && python manage.py runserver 0.0.0.0:8000
   터미널 2: cd gym_web && npm run dev
   터미널 3: ngrok http 3000
   터미널 4: ngrok http 8000

3. 앱 설정 (1분)
   - ngrok 터미널에서 URL 복사 (예: https://abc.ngrok-free.app)
   - 앱 실행 → 서버 설정
   - 복사한 URL 입력
   - 저장 및 연결 테스트

4. 완료!
   - 이제 어디서든 접속 가능
   - Wi-Fi, 4G, 5G 모두 가능

⏰ 제한사항:
   - 무료 플랜: 2시간마다 재시작 필요
   - URL이 재시작시마다 변경됨
   - 1주일 테스트엔 충분!

   - 회원, 날짜, 시간
   - 상태, 메모

5. Revenue (매출)
   - 날짜, 금액, 출처
   - 회원, 메모

6. Coach (코치)
   - 이름, 전문분야
   - 자격증, 연락처

7. WOD (수업 일정)
   - 제목, 날짜, 시간
   - 코치, 운동 내역
   - 정원, 등록 인원

8. Locker (락커)
   - 번호, 회원
   - 시작일, 종료일

9. MemberNote (회원 메모)
   - 회원, 내용
   - 작성자, 날짜
