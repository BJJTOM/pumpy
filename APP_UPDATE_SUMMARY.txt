      - 내용
      - 작성자

      - 중복 방지 (unique_together)
      - 좋아요/취소 토글
   
   ✅ Story (스토리)
      - 24시간 자동 만료
      - 조회수
      - 이미지 + 텍스트
   
   ✅ SavedPost (저장)
      - 북마크 기능
      - 나중에 보기

2️⃣ Django Serializers
   ✅ MemberBriefSerializer
   ✅ PostSerializer (with nested data)
   ✅ CommentSerializer
   ✅ StorySerializer
   ✅ SavedPostSerializer

3️⃣ Django ViewSets & API
   ✅ PostViewSet
      - CRUD
      - /posts/{id}/like/ (좋아요 토글)
      - /posts/{id}/comment/ (댓글 작성)
      - /posts/{id}/save/ (저장 토글)
   
   ✅ CommentViewSet
      - CRUD
      - 게시물별 필터링
   
   ✅ StoryViewSet
      - CRUD
      - /stories/{id}/view/ (조회수 증가)
      - /stories/cleanup_expired/ (만료 삭제)
      - 자동 24시간 만료
   
   ✅ SavedPostViewSet
      - 읽기 전용
      - 회원별 필터링

4️⃣ API Routes
   ✅ /api/posts/
   ✅ /api/comments/
   ✅ /api/stories/
   ✅ /api/saved-posts/

         📝 주요 기능 설명
═══════════════════════════════════════════════════════════════

🤖 AI 캐릭터 생성
- 실제 사진 업로드
- "AI 캐릭터 생성하기" 버튼 클릭
- 2초 후 AI 스타일로 변환 (시뮬레이션)
- 실제 AI는 API 연동 가능 (추후)

🔥 자동 칼로리 계산
- 출석 시간 기록
- 퇴관 시간 기록
- (퇴관 - 출석) * 300 kcal/시간
- 자동으로 화면에 표시

📊 실시간 출석 통계
- 연속 출석: 매일 출석했는지 체크
- 이번 달: 현재 월의 출석일 수
- 총 출석: 전체 출석 기록

📱 인스타그램 스타일 커뮤니티
- 게시물: 사진 + 캡션
- 좋아요: 토글 방식
- 댓글: 실시간 추가
- 스토리: 24시간 자동 삭제
- 저장: 북마크 기능
